How to use git

# To add this to your local machine, so you can work on it much easier do the code here:
		  $ cd
		  $ cd Documents
		  $ mkdir GithubPages
		  # This makes the folder where you put your Github Pages stuff in
		  $ cd GithubPages
		  $ git clone https://github.com/SwerveCraft/SwerveCraft.github.io
		  # This clones the repository to your computer
		  $ cd SwerveCraft.github.io

Now how to use Git commands.

# This stages the file for commit, aka lets you put up changes on the website about it, this is required
		  $ git add [file or folder]
		  # Now you can commit the file, this is how you do it
		  $ git commit [file or folder] -m '[commit messaage]'
		  # Now to update the repository ONLINE
		  $ git push
		  # Now ignore any random stuff that pops up,
		  # Username for 'https://github.com': USERNAME HERE
		  # Password for 'https://USERNAME@github.com': PASSWORD HERE
		  # Password will not show for security reasons
		  # Now your set, this is how you edit your website.
